3.2 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Cualquier tipo Input particular de formulario de HTML5 debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML5.

R= Falso. el depliegue de los input puede variar de uno a otro

b) Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor.

R= Falso.Cuando se coloca el enfoque en el campo de texto, el texto de placeholder desaparece. No se “envía” cuando el usuario hace clic en el botón Enviar (a menos que el usuario escriba el mismo texto).

c) No es necesario incluir el atributo autofocus en sus formularios.

R= verdadero.

d) Los nuevos tipos input de HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos inválidos que se envían.

R= Verdadero.

e) El tipo input range se valida automáticamente de manera intrínseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.

R= Verdadero.

f) HTML5 cuenta con validación automática para el tipo input tel.

R= Falso.Ya que dependiendo del lugar el numero telefonico puede variar

g) Si el usuario introduce un URL con un formaro inapropiado en un tipo input url, no se validará. HTML5 no valida si el URL introducido existe en realidad.

R= Verdadero

h) El elemento nav muestra un menú desplegable de hipervínculos.

R= Falso, agrupa vínculos de navegación

i) El elemento header puede usarse sólo una vez en una página.

R= Falso, se puede usar varias veces en una página incluso en el encabezado

j) Los elementos nav pueden anidarse en un elemento aside.

R= Verdadero

k) Podemos usar el elemento brk para evitar dividir palabras en lugares incorrectos.

R= Falso, el que se ocupa es wbr
