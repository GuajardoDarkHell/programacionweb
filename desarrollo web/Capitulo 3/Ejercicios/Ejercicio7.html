3.7(Creación de un formulario con autocomplote) Cree un formubrio de búsqueda simple; use un elemento input search en donde el usuario pueda introducir una consultaa de búsqueda. Use el navegador Web Firefox para probar el formulario. Escriba Junio y envíe el formulario. Después escriba una j en el demento input para ver las entradas anteriores que empiezan con j; deberá aparecer Junio debajo de el demento input. Escriba Julio y envíe el formubrio de nuevo. Ahora escriba una J en d elemento input para ver las entradas anteriores que empiezan con j; deberán aparecer Junio y Julio debajo del elemento input. Pruebe esto con sus propias consultas de búsqueda también.
